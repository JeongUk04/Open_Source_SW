\# 리눅스 프로세스 관리 명령어 정리



이 문서는 리눅스 시스템에서 프로세스를 관리하고 모니터링하는 주요 명령어인 `top`, `ps`, `jobs`, `kill`에 대해 조사하고 정리한 내용입니다.



\## 1. top (Table of Processes)

시스템의 현재 상태와 실행 중인 프로세스들의 정보를 \*\*실시간\*\*으로 모니터링하는 명령어입니다.



\### 주요 기능

\* CPU 및 메모리 점유율을 실시간으로 확인 가능

\* 실행 중인 프로세스 목록을 갱신하며 보여줌



\### 사용 예시

```bash
$ top

**단축키**
q: top 종료

k: 프로세스 종료 (PID 입력 필요)

P: CPU 사용률 순으로 정렬

M: 메모리 사용률 순으로 정렬



 **ps (Process Status)**

현재 실행 중인 프로세스의 스냅샷(순간 상태)을 출력하는 명령어.
**주요 옵션**
-e: 모든 프로세스 출력
-f: 풀 포맷(Full format)으로 상세 정보 출력
aux: BSD 문법으로 모든 사용자 및 프로세스 정보 확인
ex) $ ps -ef


**jobs**
백그라운드에서 실행 중인 작업(Job)의 목록을 보여주는 명령어.
**상태 표시**
Running: 실행 중
Stopped: 일시 중지됨
+: 현재 작업 (fg나 bg 명령의 기본 대상)
-: 이전 작업
ex) $sleep 3000 \&$ jobs


**kill**
특정 프로세스에 시그널(Signal)을 보내 종료하거나 제어하는 명령어. 주로 응답 없는 프로세스를 강제 종료할 때 사용.


**시그널 번호,이름,설명**
15,SIGTERM,정상 종료 요청 (기본값)
9,SIGKILL,강제 종료 (즉시 종료)
ex)

\# PID가 1234인 프로세스 강제 종료
$ kill -9 1234
